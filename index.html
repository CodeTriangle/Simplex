SimplexThe grid that makes sense

Simplex is the CSS grid that doesn't rely on only one type of column. You can have halves, thirds, quarters, sixths, eighths and twelfths. On top of that, it's incredibly lightweight. Whereas Bootstrap is several thousand lines of restrictive, bulky code, Simplex is a single elegant package of less than 300 lines that gives you room to customize.

Simplex was created with smaller projects in mind -- something you don't need large tracts of code for. It was designed, not to be a fully functional framework, but something you could build on depending on what you need. For larger, more intensive projects, using a more fleshed out framework may benefit you more.

The GridWhere the magic happens

Using the grid is simple. To create a container, you just need to create a div with the class="container" attribute. The grid will resize thrice based on screen size. It starts at 1200 pixels for desktops, falls to 750 pixels for tablets and the width of the screen for mobile. Rows are based on the class="row" attribute. Columns are based on fractions -- that is, class="one-half column" will produce a column taking up half of the horizontal space and class="three-fourths column" will produce a column taking three quarters of the vertical space.
